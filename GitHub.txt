Criar repositório:
 - new repository > dar nome ao repositório > colocar se é público ou privado > adicionar um read me se quiser

Clonar o repositorio para o coputador:
 - code > aba https > copiar o link > ir na pasta no seu computador e abrir o gitBash > comando git clone <link>

Ao clonar um repositório vazio, deve-se iniciá-lo com:
git init

Verificar se o servidor e seu pc estão sincronizados:
 - git status

Adicionar arquivos para commitar:
 - git add .

Commitar:
 - git commit -m "texto de descrição entre aspas"

Enviar para o repositório:
 - git push origin main
